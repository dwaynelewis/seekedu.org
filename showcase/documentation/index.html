Installing Template

After unzip Template Files, you will found HTML files and all other files. You can view this Theme in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the template. Login to your cPanel, locate the domain folder you wish to upload your files to on your server then upload all files that is available on root.

HTML Structure

This Theme has a Responsive layout with a variation of column layouts depending on device. This theme uses Twitter Bootstrap grid system(12 columns) which will help streamline any customization you wish to make. If you're unfamiliar with Twitter Bootstrap grid system, be sure to go to Twitter Bootstrap documentation, and you will see the Grid system with divs.

CSS Structure

I'm using one main CSS file (include/css/style.css) in this theme which you are free to modify depending on the customization you require. There are also few additional style sheets that you can do changes if needed.

Javascript Settings

You can do changes in Custom.js file (include/js/custom.js). There are also few additional javascript files that I do not recommend changing until you know the language very well.

Page Laod Animation

There is a plugin "Animsition" used for animating the page on load. For more information and effect please visit Animsition
